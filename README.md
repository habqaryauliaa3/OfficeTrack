# OfficeTrack

OfficeTrack adalah aplikasi berbasis Flask untuk membantu mengelola agenda dan keuangan sederhana dalam satu dashboard.

## Fitur

- Login & Register
- Dashboard agenda
- Tambah, edit, hapus agenda
- Detail agenda
- Progress tugas
- Prioritas agenda
- Status deadline
- Dark Mode
- Manajemen keuangan
- Grafik pemasukan & pengeluaran
- Riwayat aktivitas (Logs)

## Teknologi

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js

## Instalasi

Clone repository

```bash
git clone https://github.com/USERNAME/OfficeTrack.git
```

Masuk ke folder project

```bash
cd OfficeTrack
```

Install dependency

```bash
pip install -r requirements.txt
```

Jalankan aplikasi

```bash
python app.py
```

Lalu buka browser

```
http://127.0.0.1:5000
```
